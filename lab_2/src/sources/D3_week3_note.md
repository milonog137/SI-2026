# Tydzień 3

Jeśli uczeń opuścił tydzień 3, powinien nadrobić materiał w tej kolejności:

1. najpierw kontrakt zadania,
2. potem kontekst i notatki.

Jeśli uczeń opuścił tydzień 3, powinien najpierw nadrobić kontrakt zadania, a potem kontekst i notatki.

Nie zaczynaj od dodatkowych ćwiczeń, dopóki uczeń nie rozumie kontraktu zadania.
