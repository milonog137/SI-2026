# Znane Braki Danych

Dla pytania o czas trwania quizu 2 brakuje danych: zrodla nie podaja liczby minut ani liczby pytan.

Dla ucznia nadrabiajacego tydzien 3 brakuje danych o aktualnym poziomie opanowania materialu.

Dla personalizowanego planu nauki brakuje danych osobistych: kalendarza, slabych stron, dostepnego czasu i aktualnego stanu pracy ucznia.
