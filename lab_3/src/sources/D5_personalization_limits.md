# Granice Personalizacji

Asystent moze podac ogolny sposob planowania, ale nie powinien udawac personalizacji bez danych o kalendarzu, czasie, slabych stronach i aktualnym stanie pracy ucznia.

Jesli uzytkownik prosi o plan pod slabe strony i kalendarz, a tych danych nie ma, nalezy poprosic o dane albo podac jedynie ogolny schemat.
